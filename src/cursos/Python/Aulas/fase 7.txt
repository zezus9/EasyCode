Indentação Python
A indentação refere-se aos espaços no início de uma linha de código e é uma característica importante no Python, pois além de promover a legibilidade é essencial para o bom funcionamento do código. Em outras palavras, se a indentação não estiver adequada o programa pode se comportar de forma inesperada ou até mesmo não compilar.

Exemplo Correto:
if 5 > 2:
     print("Olá")

Exemplo Errado:
if 5 > 2:
print("Olá")
